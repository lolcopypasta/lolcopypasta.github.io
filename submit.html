How to Submit

We welcome contributions from the community! You can submit your LoL copypasta or ASCII art through our GitHub repository. Here's how:

  1. Click the "Create Issue" button below
  2. Sign in to GitHub (create an account if you don't have one)
  3. Fill in the issue template with your copypasta
  4. Submit and wait for approval
Create Issue

Submission Guidelines

  • Content must be League of Legends related
  • ASCII art should be properly formatted and tested
  • No offensive or inappropriate content
  • Original or commonly shared copypastas only
  • Include a brief description of your submission

Open Source

LoL Copypasta is an open source project. You can view the source code, contribute features, or report bugs on our GitHub repository.
